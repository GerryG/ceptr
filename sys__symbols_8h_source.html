241 // Semantic Type:
242 // Structures, like "float", are abstract labels.
243 // A semantic type is a label with a particular semantic application or implementation of that labelled structure.
244 // "int" is a labelled structure, "age" is a semantic implementation.

246 // Ex:
247 // /TEST_SYMBOL="t" : the root's semantic type is "semtrex_symbol_literal" and it's literal value = 't'
