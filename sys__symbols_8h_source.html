194 // Semantic Type:
195 // Structures, like "float", are abstract labels.
196 // A semantic type is a label with a particular semantic application or implementation of that labelled structure.
197 // "int" is a labelled structure, "age" is a semantic implementation.
198 //
199 // Ex:
200 // /TEST_SYMBOL="t" : the root's semantic type is "semtrex_symbol_literal" and it's literal value = 't'
201

203 // Paren-group: A group to which you apply some function.
204 // SMTP: mail from where "from is optional"
205 // Ex: /ascii-line/char='M'...etc..., (F R O M)?
206 //
207 // Curly brace is a named (or referenced) group
208 // Ex: /ascii-line/char='M'...etc..., (F R O M)?{email-address:.*}
