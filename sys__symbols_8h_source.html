229 // A semantic type is a label with a particular semantic application or implementation of that labelled structure.
230 // "int" is a labelled structure, "age" is a semantic implementation.

233 // /TEST_SYMBOL="t" : the root's semantic type is "semtrex_symbol_literal" and it's literal value = 't'
