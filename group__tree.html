Detailed Description

Semantic trees are the basic data building block for ceptr.

Copyright
Copyright (C) 2013-2014, The MetaCurrency Project (Eric Harris-Braun, Arthur Brock, et. al). This file is part of the Ceptr platform and is released under the terms of the license contained in the file LICENSE (GPLv3).

Everything in ceptr is built out of semantic trees. In ceptr all data is assumed to be meaningfull, not just structured.
